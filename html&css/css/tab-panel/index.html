解读

乍一看有点神奇, 但是仔细看下去会发现都是假象. 本例的核心样式有两个, 分别是 overflow: hidden; 和 锚点定位.

overflow: hidden

假如没有溢出隐藏, 那么会看到其实其他的内容都在可视区域的上或下侧. 整个可视区域就像是一个窗口, ul 就是躲在窗口后面的人, 就谁谁露头.

锚点定位

溢出隐藏锁定了一个小窗口, 不在窗口的人都无法被看到. 而锚点定位则负责叫人的功能. 点击标记后, 锚点定位到指定的 id, 这个人就露头了.
