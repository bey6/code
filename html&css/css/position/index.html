position: 定位

static
默认值, 默认文档流定位
relative
相对定位, 相对于正常的文档流的偏移
absolute
绝对定位, 相对于最接近的已定位元素的偏移, 可以设置 z-index
fixed
固定定位, 相对于浏览器可视窗口的偏移
