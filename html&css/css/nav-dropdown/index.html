你的美好不止于此

时间: 2019-11-25 22:30:26作者:归期

你说你很累，想找个肩膀依偎。

生活充满着辛酸，仿佛小时候对未来的憧憬在长大以后都化为了泡沫，不论是学习还是工作，都压得你喘不过气来。你说你很累，你想休息一下，想摆脱生活的苦恼，想摆脱学习的枯燥，想要到海边，到沙滩，到田野，到森林，想远离这个单一而烦恼的生活，你说你很累，想找个肩膀依偎。

你说你很孤单，想找个人来陪伴。

你走在回家的路上，望着那路边的一队又一队的小情侣，你很羡慕，你告诉我说你很羡慕他们，因为你总是一个人，你总是一个人学习，总是一个人吃饭，总是一个人工作到深夜，总是一个人守着空荡荡的房子。你说你很孤单，你想找一个懂你的人，想找你个爱你的人，想找一个能陪伴着你走过生老病死，看遍美好风景的人，你说你很孤单，你想找个人陪伴。

你说你对未来没有了希望，想要放弃自己的梦想，换来柴米油盐。

好像所有的压力在这一瞬间迸发而出，你鼻子一酸，忍不住的想要哭泣，你觉得你一事无成，你觉得梦想好像填不满生活所需，于是你说想要丢掉梦想，安稳的生活，不再去想那些虚无缥缈的东西，即使它换不来超米油盐，你对我说：未来什么的，以前计划好了，但是终究赶不上变化不是吗?

可是后来，你拍了拍身上的灰尘，又站了起来告诉我：我很累，可是我在为着我想要的生活努力，我想找的那个肩膀会会出现的，我想去的沙滩森林也会去的，我希望遇到的那个人也正在拼尽全力的向我赶来，我的梦想也会是我继续向前的动力，生活很美好，我的美好不止于此。

于是你继续向前走，这次的你眼神坚定，这次的你充满信心，这次的你对生活重新燃起了希望，这次的你，比以前任何时候的你，都要成熟。

这个你，是你吗?因为，你的美好，肯定不止于此。
